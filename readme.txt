Easily add your Tra.cm account to your Wordpress web site or blog so you can created branded short links quickly.

== Description ==

The Tra.cm plugin allows you to easily add your company's branded URL shortener links to your Wordpress blog or web site.

== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "Tra.cm".
3. Click install.
4. Click activate.
5. Click settings to configure your API credentials.
