Easily add your tra.cm account to your Wordpress web site or blog so you can create branded short links quickly. If you have a paid account, you can use your company's own branded domains to increase brand recognition and will receive access to advanced reporting/analytics that allow you to see how your social and in-app campaigns are performing.

== Description ==

The tra.cm plugin allows you to easily add your company's branded URL shortener links to your Wordpress blog or web site.

== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "tra.cm".
3. Click install.
4. Click activate.
5. Click settings to configure your API credentials.
